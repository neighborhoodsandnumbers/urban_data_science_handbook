# Complex Systems

## A Computational Approach

This book is a comprehensive, technical graduate textbook in complex systems. It is organized as a progression from the mathematics of single systems to interacting systems to adaptive systems, with Python woven in as the medium of exploration rather than an afterthought.

Each chapter combines three modes of engagement:

- **Analytical exposition** — the mathematical foundations, stated precisely
- **Worked examples** — connecting theory to concrete problems
- **Computational labs** — Python implementations that make the abstractions tangible

### Who this book is for

Graduate students and researchers who want a rigorous, computation-forward treatment of complexity — from dynamical systems and bifurcation theory through networks and game theory to evolution, learning, and emergence.

### How to use this book

The three parts build on each other:

1. **Single Systems** — dynamical systems, stability, chaos, information theory
2. **Interacting Systems** — networks, game theory, coupled oscillators, epidemics
3. **Adaptive Systems** — evolution, learning, agent-based models, emergence

Code is not supplementary. The computational labs are where intuition is built.
